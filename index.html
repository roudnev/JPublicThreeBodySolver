Java code for solving Faddeev equations

The code has two parts: a configuration editor and a computational kernel. Current version of the code allows users investigate quantum three-body systems numerically below three-body threshold. This includes bound states and particle-dimer scattering calculations.

You may find Java binaries in the corresponding /dist/ subdirectories, and the brief description is in /doc. You are very welcome to contact the author at v.roudnev[AT]gmail.com with any enquiries concerning the code functionality and the desired features.

There are many ways to improve this code and extend its functionality, which can be an interesting experience for a student who need a challenging project in scientific computations.

If you are going to use the code in your real scientific calculations, there are definitely some features that look missing or undocumented. Do not hesitate to contact me, we shall see what can be done together.

Yours, Vladimir Roudnev
